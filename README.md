## 簡介
Broken Hand (bh) 是一個簡單的直譯式程式語言，支援基本運算、條件判斷、迴圈、函式定義與呼叫，以及布林與數字型別。
語法輕量，方便初學者理解編譯器原理。

---

## 語法特色
- 指派使用 `=`
- 輸出使用 `print`
- 條件判斷使用 `? (condition) { ... }`
- 迴圈使用 `@ (condition) { ... }`
- 函式定義使用 `f funcname(params) { ... }`
- 函式回傳使用 `r expression`
- 支援布林值 `T` 與 `F`
